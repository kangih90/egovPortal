# egovPortal
전자정부 프레임워크 egovFrameWork Template Project(Portal Site) 초기 프로젝트 세팅 - 포털사이트 기능 구현시 필수적인 부분만 사용 가능하도록 경량화 된 실행환경 제공
* 제공기능: 포털 공지사항 관리, 사용자관리, 권한관리, 포털 초기화면 관리, 포털 공지사항 관리, 포털 FAQ관리, 포털 행사참가 관리
